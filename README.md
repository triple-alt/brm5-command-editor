# brm5-command-editor

Yes it's made with Claude, due to my lack of time to actually make it from scratch.

An IDE for writing commands, more specifically for triggers.

Functionality:

- Common IDE functionality (tab moving and renaming, import and export)
- Supports most commands.
- Lists all materials and props from the asset browser.
- Command list that explains what each command and parameter means.
- (Soon) Tips and tricks menu where community written guides are there for you to learn advanced methods for triggers.

Message m_smi / triple_xx with any guides you would want added, including whole examples.
